How to set up settings

The following properties can be and must be used in a setting object.

  • name (string): name of trigger, should be unique
  • subject (string): subject template of alert email
  • emailBodyType (enum):
    • SimpleHtml = HTML string defined in GS file
    • GoogleDoc = a Google Doc
  • emailBody (HTML string): HTML template in case of EmailBodyType.SimpleHtml
  • googleDocId (string/id): Google Doc id to use as email template in case of EmailBodyType.GoogleDoc
  • urlToCheck (string/URL): URL of webpage to check
  • emailAddresses (JSON array of strings): email addresses to alert in case of webpage change
  • checkIntervalUnit (enum):
    • Hour = schedule will be hour based
    • Minute = schedule will be minute based
  • checkInterval (int): schedule interval
    • Hour = can be only 1, 2, 4, 6, 8 or 12
    • Minute = can be only 1, 5, 10, 15 or 30
  • stripHtml (bool): should strip out HTML tags from the webpage and check only the content (true) or check the original page as full HTML (false)
